Clean Code 목차

  • 작가

    • Robert C. Martin

  • 출판일

    • 2008년

  • 감상평에 대한 AI그림

    • 이 책을 읽고 나니, 단순히 코드를 '돌아가게' 만드는 것이 아니라, 읽기 쉽고 유지보수하기 좋은 코드를 짜는 것이 얼마나 중요한지 깨달았다. 특히, 함수의 길이를 짧게 유지하고, 명확한 네이밍을 하는 것이 코드 가독성을 높이는 핵심이라는 점이 인상적이었다. 앞으로 프로젝트를 할 때 항상 이 원칙들을 떠올리면서 코딩해야겠다.

  • 감상평에 대한 AI평가

    • Clean Code" 감상평 → 분석적, 실용적 코드 품질을 향상시키려는 실용적 태도가 엿보인다. 명확한 원칙을 배우고 이를 실천하려는 적극적인 학습 태도를 보여줌.
